# 🧧 Red Pocket

An interactive web experience where opening a virtual red pocket reveals a fortune in a cinematic, fate-driven world.

## ✨ Overview

**Red Pocket** is a single-page static website built with HTML, CSS, and JavaScript.  
It combines traditional red pocket symbolism with animated visuals, atmospheric UI, and fortune-reveal interactions.

The experience is designed to feel:
- mystical
- immersive
- playful
- visually rich

## 🎮 Features

- 🧧 Virtual red pocket opening interaction
- 🔮 Fortune reveal overlay
- ✨ Animated visual effects
- 🎨 Custom art direction and Chinese-inspired styling
- 🖱️ Interactive cursor and UI feedback
- 📱 Responsive browser experience
- 🌐 Fully static and GitHub Pages friendly

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Google Fonts
